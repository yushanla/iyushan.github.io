sublime安装SFTP插件

##sublime text 3 安装SFTP插件

Sumblime Text 3，按下Ctrl+Shift+P，调出命令面板，输入install Package，回车。

等待出现安装面板，输入sftp，回车。

在本地新建一个文件夹，文件名为event

打开sublime，在文件下打开文件夹，选择刚才创建的event文件夹

右键点击sublime左侧出现的event，按下图选

此时会出现一个json文件，可以直接复制下面的内容到里面
(使用linux新建的一个名rules的普通用户)

-

{
// The tab key will cycle through the settings when first created
// Visit http://wbond.net/sublime_packages/sftp/settings for help

// sftp, ftp or ftps
"type": "sftp",

"save_before_upload": true,
"upload_on_save": true,
"sync_down_on_open": true,
"sync_skip_deletes": false,
"sync_same_age": true,
"confirm_downloads": false,
"confirm_sync": true,
"confirm_overwrite_newer": false,

"host": "192.168.203.216",
"user": "rules",
"password": "Lthis.5",
"port": "22",

"remote_path": "/home/snort/perseus6x/psconf/apd-config/event",
"ignore_regexes": [
    "\\.sublime-(project|workspace)", "sftp-config(-alt\\d?)?\\.json",
    "sftp-settings\\.json", "/venv/", "\\.svn/", "\\.hg/", "\\.git/",
    "\\.bzr", "_darcs", "CVS", "\\.DS_Store", "Thumbs\\.db", "desktop\\.ini"
],
//"file_permissions": "664",
//"dir_permissions": "775",

//"extra_list_connections": 0,

"connect_timeout": 30,
//"keepalive": 120,
//"ftp_passive_mode": true,
//"ftp_obey_passive_host": false,
//"ssh_key_file": "~/.ssh/id_rsa",
//"sftp_flags": ["-F", "/path/to/ssh_config"],

//"preserve_modification_times": false,
//"remote_time_offset_in_hours": 0,
//"remote_encoding": "utf-8",
//"remote_locale": "C",
//"allow_config_upload": false,
}

配置完json文件，再右键点击sublime左侧event，进入 SFTP/FTP，选择Sync Remote -> Local,即可将测试设备上的规则文件下载到本机的event文件中。

选择 YES

在本地修改event文件下的xml规则文件，保存文件后，sublime会自动上传改动到服务器

Done!
