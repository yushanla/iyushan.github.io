windows10下使用linux(非虚拟机)

在今年的 Build 2016 上，微软向全世界介绍了他们还处于 Beta 阶段的 Windows 下的 Linux 子系统（WSL），它可以让开发者们在 Windows 10 下通过 Bash shell 运行原生的 Ubuntu 用户态二进制程序。如果你参与了 Windows Insider 计划，你就可以在最新的 Windows 10 年度升级版的 Insider 构建版中体验这个功能了。

之前看到别人直接在windows下玩linux，很是炫酷，今天才终于在自己笔记本上试了试，过程很顺利，效果很满意，现在将过程记录下来，以备将来忘记时查看，同时也给那些想折腾的同学提供一些经验。

前提是必须是win10系统，最好将版本升级到最新版，至于需不需要用户加入 Insider，因为我之前就已经加入了，所以不知道需不需要这个，有待验证。

首先，在win10设置->更新与安全->针对开发人员->选择开发人员模式

然后，在控制面板中，进入启用或关闭windows功能，选上 适用于linux的windows的子系统，之后需要重启一下

最后，在cmd命令行中键入 bash ，之后就等待安装就可以了，等安装完成就进入了linux shell下！

以后进入linux，直接输入bash即可，想退出就键入exit。

安装完成后需要注意的几点：

1.关于如何进入root用户下

在linux下输入sudo passwd root，然后输入两遍密码即可

2.关于linux系统所在路径

C:\Users[你的用户名]\AppData\Local\lxss

有隐藏目录，如果想一级一级往下找linux路径，别忘了关闭 隐藏目录 功能

Done!
